Fire and Smoke Detection using YOLOv8

This project focuses on developing a real-time fire and smoke detection system using the YOLOv8 deep learning model. The system detects fire hazards quickly using computer vision to improve safety and emergency response.

The system was trained on a custom dataset containing images of fire and smoke. Using YOLOv8 object detection, the model identifies fire hazards in real time and provides quick alerts.

Detecting fire in real-time environments requires accurate training and fast detection speed. The challenge was ensuring the model performs well under different lighting and environmental conditions.

By training YOLOv8 with a custom dataset and optimizing detection parameters, the system achieved reliable fire and smoke detection with fast response time.

Key Features

  • Real-time fire detection
  • YOLOv8 deep learning model
  • Custom dataset training
  • Computer vision based monitoring
  • Improved emergency response detection
